# Push to GitHub

Your repo: **https://github.com/ibrahimmarvinochieng-beep/swift.git**

## 1. Install Git (if not installed)

Download: https://git-scm.com/download/win

## 2. Open PowerShell in this folder and run:

```powershell
cd c:\Users\ADMIN\Downloads\swift_project

git init
git add .
git commit -m "Initial commit: Swift Event Intelligence Platform"

git remote add origin https://github.com/ibrahimmarvinochieng-beep/swift.git
git branch -M main
git push -u origin main
```

## 3. When prompted

- Sign in to GitHub (browser will open)
- Or use a Personal Access Token as password

Done! Your code will be at https://github.com/ibrahimmarvinochieng-beep/swift
